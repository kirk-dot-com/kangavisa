# KangaVisa GovData Pilot Program Pack v1.0

## 1. Purpose

This document outlines a **90-day pilot program** for KangaVisa GovData designed for:

- Australian Government Departments  
- Embassies / High Commissions  
- Policy and migration stakeholders  

Goal:
Validate value → demonstrate insights → convert to paid subscription

---

## 2. Pilot Overview

**Duration:** 60–90 days  
**Cost:** Free or low-cost (strategic pilot)  
**Output:** Live dashboard + insights + final report  

---

## 3. Pilot Objectives

### For Client
- Understand pre-lodgement migration behaviour
- Identify early risk signals
- Test usability of dashboard + datasets

### For KangaVisa
- Validate data usefulness
- Refine datasets
- Secure long-term contract

---

## 4. Pilot Scope

### Included

- GovData dashboard access  
- 1–2 country focus areas  
- Visitor visa (Subclass 600) dataset  
- Monthly insights briefing  
- Final pilot report  

### Optional Add-ons

- API access (limited)  
- Additional visa subclasses  
- Custom metrics  

---

## 5. Data Delivered

### Core Datasets

1. Visitor Readiness Dataset  
2. Risk Indicator Dataset  
3. Demand & Behaviour Dataset  

All data is:
- Aggregated  
- De-identified  
- Consent-based
